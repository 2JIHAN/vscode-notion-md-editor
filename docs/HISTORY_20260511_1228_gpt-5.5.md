### Callout Markdown 표준 결정

Notion 스타일 callout은 순수 Markdown blockquote가 아니라 Notion Enhanced Markdown에 가까운 XML-like 태그를 사용하기로 했다.

표준:

```md
<callout icon="✅" color="green_bg">
**성공 기준**

업로드가 정상 완료된 상태
</callout>
```

## 2026-05-14

### 모듈 분리와 Phase 1 착수

`src/extension.js` 단일 파일(~19KB)을 네 개 모듈로 분리했다.

- `src/markdown.js` AST 기반 parser, serializer, HTML renderer 순수 함수
- `src/callouts.js` callout 정의 상수
- `src/webview.js` preview, WYSIWYG webview HTML 템플릿
- `src/extension.js` VS Code 통합만 담당

분리 목적은 markdown round-trip 동작을 Node 단위 테스트로 검증할 수 있게 하는 것이다.

### Frontmatter 보존

YAML frontmatter(`---` 구분)를 첫 블록으로 인식하고 별도 블록 타입(`frontmatter`)으로 보존한다. WYSIWYG에서는 본문 영역에 렌더링하지 않고, 저장 시 원본 frontmatter를 본문 앞에 다시 붙인다.

### 라운드트립 테스트 추가

`test/markdown.test.js`에 13개 테스트를 작성했고 모두 통과한다. heading, list, code block, callout(중첩 블록 포함), frontmatter, HTML escape, inline emphasis를 다룬다. `docs/fixtures/`에 basic, callouts, frontmatter 세 문서를 두고 parse → serialize → parse 비교로 AST 보존을 확인한다.

### Inline emphasis 확장

기존 renderer는 bold, inline code, link만 처리했다. `*text*` 기울임을 추가했다.

### WYSIWYG 편집 UX 개선

사용자 피드백 기반 버그 5건을 정리했다.

1. **라이트, 다크, 자동 테마 토글** webview 상단에 토글 버튼 추가. `vscode.setState`로 선택 상태 보존. Notion 스타일 팔레트 적용.
2. **Cmd+Z, Cmd+Shift+Z** webview에서 `document.execCommand('undo'/'redo')`로 `contenteditable` native undo에 위임. 매 250ms `WorkspaceEdit` replace는 그대로 유지하되 webview 내부 편집 이력을 따른다.
3. **`` `text` `` 자동 변환** `input` 이벤트에서 패턴 감지 후 `<code>` 요소로 치환. 커서 위치 보존.
4. **Cmd+E** inline code 토글. 선택 영역이 있으면 감싸고, 없으면 빈 `<code>` 삽입. 이미 code 안이면 해제.
5. **삼중 백틱 + Enter** `` ``` `` 또는 `` ```lang `` 단독 문단에서 Enter를 누르면 `<pre><code data-lang>`으로 치환.

추가로 Cmd+B, Cmd+I 단축키도 wired up.

### Markdown serializer 보강

`pre code[data-lang]` 처리 추가. WYSIWYG에서 만든 fenced code block이 markdown으로 직렬화될 때 언어 태그가 보존된다.

### 노션 스타일 다듬기

- 기본 테마를 Light로 변경. 토글 순서 light, dark, auto.
- 인라인 코드 색상을 노션 hex `#EB5757` (light) 또는 `#FF8B8B` (dark)로 통일.
- 코드 블록 배경을 노션 light 팔레트 `rgba(247, 246, 243, 1)`로 조정.
- 본문 폰트 system sans-serif 명시. 한글은 Apple SD Gothic Neo, Malgun Gothic으로 fallback.
- monospace 폰트는 Menlo 우선, 한글은 D2Coding fallback. Nerd Font 변형은 사용하지 않는다.
- 백틱 3개 입력 즉시 코드 블록으로 변환 (Enter 불필요).
- 코드 블록 내부에서 Enter는 새 블록 생성 대신 줄바꿈 삽입.
